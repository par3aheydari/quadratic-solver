# Quadratic Solver

A simple **HTML/CSS/JS** project to solve quadratic equations and display an interactive graph.

---

## Features

- Solve quadratic equations (`axÂ² + bx + c = 0`)  
- Show **real and complex roots** separately  
- Interactive **parabola graph**  
- Modern **red/black theme** with animated background  
- Fully client-side, no backend needed  

---

## How to Use

1. Open `index.html` in any modern browser.  
2. Enter the coefficients `a`, `b`, and `c`.  
3. Click **Solve**.  
4. Roots will be displayed below the inputs.  
5. The parabola graph will update automatically.
